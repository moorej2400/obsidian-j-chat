# Obsidian J Chat

Vault-aware AI chat side panel for Obsidian.

## Development

```bash
npm install
npm run test
npm run build
```

The plugin stores user settings through Obsidian plugin data. Do not commit local plugin data, API keys, `.env*`, `.claude/`, `.codex/`, or `.teamwork/`.
